# Depreciator

A simple Python depreciation calculator.

Currently supports:
- Straight-line depreciation (single assets)
- Double declining balance
- Sum of years' digits
- Baskets of assets with varied depreciation methods

---

## Inputs

- Initial cost
- Salvage value
- Useful life
- Depreciation method

**Output:**
- Formatted depreciation schedule  
- Years are 1-indexed in accordance with financial and accounting convention

---

## Requirements

- Python 3.14
- No external dependencies

---

## Usage

Run from the project directory:

```bash
python depreciator.py
